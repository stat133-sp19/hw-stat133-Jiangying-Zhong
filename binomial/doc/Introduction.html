library(binomial)

Binomial

The package "binomial" is a minimal implementation for calculating probabilities of a Binomial random variable.

check the private attribution

The first step is to check the private attributions of a binomial random varible. In general, the trials must be a positive number,the prob must be a positive number between 0 and 1, and the success must be a positive number less than the trials. Addition to this, the prob must be 1 length.

auxiliary functions for summary measures

Then, you can use the auxiliary functions to simulate the basic attributes of the binomial random variable. When X is a random variable that has a binomial distribution with n trials and probability of success p, we write X ∼ Bin(n, p). The expected value or mean of a binomial distribution is: np. The variance is given by: np(1 − p). For 0 < p < 1, the most likely number of success in n independent trials with probability p of success on each trial is m, the greater integer less than or equal to np + p: m = int(np + p). The skewness of a binomial random variable can be calculated as: skewness =(1 − 2p)/sqrt(np(1 − p)). For a binomial random variable, its kurtosis can be obtained as: kurtosis = (1 − 6p(1 − p))/np(1 − p).
